# TaskFlow

> A modern team task management application built with Next.js, TypeScript, and Prisma.

## Overview

TaskFlow is a collaborative task management platform designed for high performing engineering teams. It helps teams track issues, ship faster, and stay organized with a clean, intuitive interface.

#### Authentication & Authorization
- Secure email/password authentication with bcrypt hashing
- Google OAuth integration
- Protected routes with NextAuth.js
- Form validation with Zod and React Hook Form
- Clean, modern dark-themed UI
- Real-time form validation and error handling
- Automatic session management (30-day expiry)
